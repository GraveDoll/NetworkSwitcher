# NetworkSwitcher
ネットワークアダプターを切り替えるツール

## Description
登録したネットワークアダプターの有効・無効を切り替えるツールです。
タスクトレイに常駐し、素早くネットワークを切り替えることができます。

## Requirements
Windows 7 または Windws10
